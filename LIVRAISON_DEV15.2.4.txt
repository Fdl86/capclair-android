-------------------------

- Le document est construit à partir d'un instantané structuré et stable.
- Les distances sont arrondies au NM entier le plus proche.

- La première version est limitée à 8 branches avec avertissement au-delà.
- Android partage le PDF comme fichier binaire via le plugin existant étendu en Base64.
- Les exports GPX et JSON conservent leur fonctionnement UTF-8 existant.
- Le web/PWA télécharge le même PDF avec un Blob local.
- Le gabarit PDF est inclus dans l'APK et dans le cache PWA.

1. Vider le dossier local de la branche APK en conservant uniquement .git.
2. Copier tout le contenu du ZIP dans ce dossier.
3. Ouvrir GitHub Desktop et vérifier que la branche active est main.
4. Créer le commit : DEV15.2.4 - Export PDF Log nav
5. Pousser sur GitHub.
6. Attendre le dernier run vert Android Debug APK.
7. Télécharger uniquement : cap-clair-dev15-2-4-debug-apk
8. Installer l'APK par-dessus la version précédente.
9. Vérifier dans l'application : DEV15.2.4 et le hash court du build.
10. Ne désinstaller qu'en cas de problème confirmé de versionCode ou de signature.
